# DSC 106 Final Project: Understanding Hurricanes

This website presents our scroll-driven exploration of hurricane science, impacts, and preparedness. Our project implements a hurricane anatomy diagram, build your own hurricane simulation, map of clickable real-time hurricane popups, multiple interactive, insightful charts, and a call to action infographic.

By Cedric Jeng, Henry Liu, Siddhartha Sarvepally, Maggie Zhang
