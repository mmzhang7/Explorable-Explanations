# LaTeX Resume Passage

This repository includes a LaTeX-formatted resume passage in `resume_latex_passage.tex` that can be used to describe the "Understanding Hurricanes" project in a resume or CV.

## Usage

Copy the contents of `resume_latex_passage.tex` and paste it into your LaTeX resume document in the projects section. This assumes your resume uses the following LaTeX commands:
- `\resumeProjectHeading{}{}` - for the project header
- `\resumeItemListStart` and `\resumeItemListEnd` - for the bullet point list
- `\resumeItem{}` - for individual bullet points

## Customization

You may want to adjust:
- The date range (currently set to "May 2024 -- June 2024")
- The project URL (currently points to `https://mmzhang7.github.io/Explorable-Explanations/`)
- Specific accomplishments based on your role in the project
- Technology stack emphasis based on what you want to highlight

## Project Overview

This passage describes a data visualization project that:
- Built an interactive scroll-driven website about hurricane science
- Used D3.js and Scrollama for dynamic visualizations
- Analyzed historical hurricane data spanning 100+ years
- Created educational content with actionable preparedness information
